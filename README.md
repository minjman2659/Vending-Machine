# Vending-Machine

심심해서 만들어본 Vending-Machine by TS <br />
자판기에서 사용자가 원하는 음료수들을 얻기까지의 메커니즘을 TS로 도식화

## 전제조건

1. 사용자가 사용가능한 결제수단 <br />
   a. 현금 : 100원 / 500원 / 1,000원 / 5,000원 / 10,000원권 사용가능 <br />
   b. 카드 : 카드결제 가능 <br />

2. 구매 가능한 음료수 <br />
   a. 콜라 : 1,100원 <br />
   b. 물 : 600원 <br />
   c. 커피 : 700원 <br />
